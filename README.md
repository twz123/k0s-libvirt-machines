# k0s-libvirt-machines

Use [OpenTofu] and [libvirt] to bring up Virtual machines for [k0s].

[OpenTofu]: https://opentofu.org/
[libvirt]: https://libvirt.org/
[k0s]: https://k0sproject.io/
